# Yazi

Terminal file manager launched via tmux floating popup.

## Keybinding

- Open Yazi: `Prefix + CTRL + y` (tmux popup, 90% width/height)

## Config

- Config location: `~/.config/yazi/yazi.toml`
- Show hidden files: enabled

## Settings

| Setting | Value |
|---------|-------|
| `show_hidden` | `true` |

# Ghostty Cheatsheet

## Keybindings

### General

- Reload config: `Super+b` then `r`
- Close surface: `Super+b` then `x`
- New window: `Cmd+s` then `n`

### Tabs

- New tab: `Super+b` then `c`
- Previous tab: `Super+b` then `,`
- Next tab: `Super+b` then `.`
- Go to tab 1-9: `Super+b` then `1-9`

### Splits

- Split right: `Super+b` then `\`
- Split down: `Super+b` then `-`
- Navigate to split: `Super+b` then `h/j/k/l` or arrow keys
- Zoom split: `Super+b` then `z`
- Equalize splits: `Super+b` then `e`

### Copy

- Copy: `Cmd+C` (also works in tmux copy mode)
- Copy on select: enabled (copies to clipboard automatically)

## Config

- Config location: `~/.config/ghostty/config`
- Font: JetBrainsMono Nerd Font (size 14)
- Cursor: bar, blinking, green (`#A3BE8C`)
- Theme: Solarized Dark
- Shell: launches into tmux automatically (`tmux attach || tmux`)
- Titlebar: hidden
- Window state: saved across restarts
